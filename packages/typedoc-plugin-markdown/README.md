# typedoc-plugin-markdown

## Overview

By default [TypeDoc](https://typedoc.org) will output API documentation as HTML. This plugin replaces the default HTML theme with a built-in Markdown theme and exposes some additional options.

Since many tools and platforms support Markdown, the documentation generated by this plugin can be easily integrated into different ecosystems, such as code repositories, Wikis and static site generators.

## Installation

```shell
npm install typedoc typedoc-plugin-markdown --save-dev
```

## Features

- [CommonMark](https://spec.commonmark.org/0.31.2/), [GFM](https://github.github.com/gfm/) and [MDX](https://mdxjs.com/) compliant.
- Works alongside other TypeDoc plugins.
- Configurable output file structure.
- Configurable UX options
- [Frontmatter](https://typedoc-plugin-markdown.org/plugins/frontmatter) and [remark](https://typedoc-plugin-markdown.org/plugins/remark) utility plugins available.
