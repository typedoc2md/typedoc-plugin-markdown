# Options

This page provides an overview of all available options for the plugin.
It includes both the
core options provided by TypeDoc (where supported) and the additional options
added by this plugin.

## TypeDoc Options

TypeDoc exposes many options, most of which this plugin supports while others are HTML-only and ignored.

### Output

<Callout type="important">
  Output options control TypeDoc's generated output. These options mainly apply
  to TypeDoc's default HTML renderer, but this plugin supports or extends a
  subset.
</Callout>

_The options listed here are the output options that this plugin supports or extends._

{/* TYPEDOC_OUTPUT_OPTIONS_START - This block is dynamically generated */}

| Option Name                                                         | Description                                                               |
| ------------------------------------------------------------------- | ------------------------------------------------------------------------- |
| [outputs](./options/output#outputs)                                 | Specifies the outputs which should be rendered by TypeDoc.                |
| [out](./options/output#out)                                         | Specifies the location the markdown should be written to.                 |
| [router](./options/output#router)                                   | Selects the router used to determine the file and folder structure.       |
| [lang](./options/output#lang)                                       | Sets the language of the translations used when generating documentation. |
| [cleanOutputDir](./options/output#cleanoutputdir)                   | Used to prevent TypeDoc from cleaning the output directory.               |
| [navigation](./options/output#navigation)                           | Determines how the navigation JSON structure will be built.               |
| [includeHierarchySummary](./options/output#includehierarchysummary) | Specifies whether or not to generate a dedicated hierarchy page.          |

{/* TYPEDOC_OUTPUT_OPTIONS_END */}

### Conversion

<Callout type="important">
  Conversion options control how TypeDoc analyzes the source code and builds the
  reflection models. They are theme-agnostic and are not affected by this
  plugin.
</Callout>

_The option groups listed here link directly to the TypeDoc documentation website._

{/* TYPEDOC_CONVERSION_OPTIONS_START - This block is dynamically generated */}

| Option Group                                                              | Description                                                        |
| ------------------------------------------------------------------------- | ------------------------------------------------------------------ |
| [Configuration](https://typedoc.org/documents/Options.Configuration.html) | Options that control where TypeDoc reads its configuration from.   |
| [Input](https://typedoc.org/documents/Options.Input.html)                 | Options that control what files TypeDoc processes.                 |
| [Comments](https://typedoc.org/documents/Options.Comments.html)           | Options that control how TypeDoc parses comments.                  |
| [Organization](https://typedoc.org/documents/Options.Organization.html)   | Options that control how content is structured on generated pages. |
| [Validation](https://typedoc.org/documents/Options.Validation.html)       | Options that control how TypeDoc validates your documentation.     |
| [Other](https://typedoc.org/documents/Options.Other.html)                 | Options which don’t fit elsewhere.                                 |

{/* TYPEDOC_CONVERSION_OPTIONS_END */}

## Plugin Options

In addition to the TypeDoc core options, this plugin provides its own options.
They can be viewed as a subset of output-related options specific to this plugin's functionality.

{/* PLUGIN_OPTIONS - This block is dynamically generated */}

### File

  File options are used by the plugin’s{' '}
  <OptionLink type="output" name="router" /> to configure file output details.

| Option Name                                                 | Description                                              |
| ----------------------------------------------------------- | -------------------------------------------------------- |
| [fileExtension](./options/file#fileextension)               | Specify the file extension for generated output files.   |
| [entryFileName](./options/file#entryfilename)               | The file name of the entry page.                         |

| [excludeScopesInPaths](./options/file#excludescopesinpaths) | Exclude writing @ scope directories in paths.            |
