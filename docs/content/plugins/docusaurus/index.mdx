# Docusaurus

## Overview

The Docusaurus integration lets you generate TypeScript API reference docs as Markdown that fits into a Docusaurus docs site.

## Features

- Generates Docusaurus-compatible Markdown from TypeDoc projects.
- Supports both Docusaurus-managed and standalone TypeDoc generation.
- Generates sidebar data for Docusaurus docs navigation.
